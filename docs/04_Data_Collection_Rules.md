# Data Collection Rules

## Purpose

This document standardizes how brand, product, market, and B2B data should be collected and maintained.

## Preferred Sources

- ★★★★★ Official Website
- ★★★★★ Official Product Pages
- ★★★★★ Official Social Accounts
- ★★★★ Authorized Retailers
- ★★★ Industry Media
- ★★ Reddit / Community
- ★ Other References

## Core Rules

- Never fabricate data.
- Unknown is always better than guessing.
- Every verified field should include source URLs.
- Homepage URL only verifies brand existence unless a specific field is visible on that homepage.
- Product records require real product-level sources.

## Field Verification

Each verified field should be supported by a source URL when possible:

- Website
- Product category
- Product name
- Price
- Size range
- Fabric or composition
- Contact page
- Social account
- Manufacturing signal

## Unknown Values

Use `Unknown` when a field cannot be verified. Do not create fake emails, fake prices, fake social handles, fake physical addresses, fake SKUs, or guessed manufacturing locations.
