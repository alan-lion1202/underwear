# Underwear Intelligence Project

## Project Goal
Build an LGBTQ+ underwear industry intelligence platform for product research, competitor analysis, B2B customer intelligence, and sample development.

## Data Rules
- Never fabricate data.
- If a field cannot be verified, use "Unknown".
- Every verified field should include a source URL.
- Do not invent brands, products, prices, emails, addresses, or manufacturing locations.
- Prefer official brand websites and official social accounts.

## UI Rules
- Keep tables compact.
- Minimize horizontal scrolling.
- Use expandable detail rows.
- Keep Brand/Product columns sticky.
- Include back button and breadcrumbs on detail pages.

## Development Rules
- ChatGPT provides research, database design, and product strategy.
- Codex focuses on implementation.
- Do not redesign the entire project unless explicitly requested.
- Do not add authentication, Supabase, or AI automation unless explicitly requested.
