# Underwear Intel Dashboard

Next.js dashboard MVP for LGBTQ+ underwear market intelligence, manual product research, and B2B customer development.

## Stack

- Next.js 15
- TypeScript
- App Router
- CSV seed/import/export workflow

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://127.0.0.1:3000/dashboard`.
